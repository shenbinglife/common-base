# common-base
最基本的通用工具包，提供基本的异常结构，通用注解，工具类等

## Maven地址  
```$xslt
<dependency>
    <groupId>io.github.shenbinglife</groupId>
    <artifactId>common-base</artifactId>
    <version>1.0.0</version>
</dependency>
```
